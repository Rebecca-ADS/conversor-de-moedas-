# conversor-de-moedas-
curso de Back End 
Descrição do Projeto

Este projeto consiste em um Conversor de Moedas desenvolvido em Java, com foco em back-end, permitindo a conversão entre diferentes moedas a partir de taxas de câmbio atualizadas.

O sistema foi projetado para simular uma aplicação real de back-end, aplicando conceitos como organização de código, orientação a objetos, boas práticas, tratamento de erros e consumo de dados externos.

Objetivo

O principal objetivo do projeto é:

Praticar Java para back-end

Trabalhar com lógica de conversão monetária

Simular o consumo de serviços de câmbio

Criar uma base para futura evolução em API REST

Funcionalidades

Conversão entre moedas (ex: BRL, USD, EUR)

Entrada de valor pelo usuário

Seleção da moeda de origem e destino

Cálculo automático da conversão

Exibição do valor convertido

Validação de entradas inválidas

 Conceitos Aplicados

Programação Orientada a Objetos (POO)

Estruturação de camadas

Lógica de negócio no back-end

Tratamento de exceções

Boas práticas de código Java
